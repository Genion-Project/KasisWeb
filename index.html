Panduan lengkap untuk memulai menggunakan KASIS

Panduan Instalasi

Windows

Opsi 1: Installer (Recommended)

  1. Download file installer KasisAppSetup.exe
  2. Jalankan file installer dengan double-click
  3. Ikuti wizard instalasi hingga selesai
  4. Buka aplikasi KASIS dari Start Menu atau Desktop

Opsi 2: Portable (ZIP)

  1. Download file KASIS-Windows.zip
  2. Extract file zip ke folder pilihan Anda
  3. Jalankan file KASIS.exe dari folder hasil extract

Linux

  1. Download file KASIS.zip dari halaman download
  2. Extract file zip ke folder pilihan Anda
  3. Buka terminal dan navigasi ke folder hasil extract
  4. Berikan permission execute dengan chmod +x KASIS
  5. Jalankan aplikasi dengan ./KASIS

Android

  1. Download file KASIS.apk
  2. Aktifkan "Install from Unknown Sources" di Settings
  3. Buka file APK dan tap Install
  4. Buka aplikasi KASIS dari app drawer

Cara Penggunaan

1

Setup Awal

Setelah instalasi, buat akun admin pertama dengan memasukkan nama sekolah, alamat, dan data administrator

2

Mengelola Kas

Masuk ke menu Kas untuk melihat saldo, tambah pemasukan atau pengeluaran. Sistem akan otomatis menghitung saldo akhir

3

Catat Pelanggaran

Di menu Pelanggaran, pilih siswa, jenis pelanggaran, dan point akan otomatis ditambahkan. Export laporan dalam format PDF atau Excel

4

Generate Laporan

Buat laporan bulanan atau tahunan untuk kas dan pelanggaran. Laporan dapat dicetak atau disimpan sebagai file
